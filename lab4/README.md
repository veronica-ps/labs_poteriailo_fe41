Інструкція до програми: <br>
За допомогою слайдерів Amplitude, Frequency та Phase ви можете змінювати вигляд синьої пунктирної лінії (чистої гармоніки).

Налаштування шуму: Слайдери Noise Mean та Noise Covariance регулюють зміщення та інтенсивність шуму.

**При зміні параметрів самої гармоніки шум залишається незмінним (не генерується наново), що дозволяє бачити вплив фільтрації на один і той самий набір перешкод.**

Управління відображенням (Чекбокси):
* Show Noise — вмикає/вимикає помаранчеву лінію зашумленого сигналу.
* Show Filtered — вмикає/вимикає фіолетову лінію відфільтрованого сигналу.

Слайдер Filter Cutoff дозволяє регулювати частоту зрізу фільтра Баттерворта. Налаштуйте його так, щоб фіолетова лінія була максимально близькою до чистої синьої гармоніки.

Натисніть кнопку «Reset» для миттєвого повернення до початкових параметрів системи, визначених у завданні.
