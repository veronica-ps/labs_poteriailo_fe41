### 1. Бічна панель
Тут знаходяться всі елементи управління відображенням даних:
* **Вибір індексу (Dropdown):** Дозволяє обрати один з трьох вегетаційних індексів для аналізу (VCI, TCI або VHI). Усі графіки та таблиці миттєво перемикаються на обраний показник.
* **Вибір області (Dropdown):** Список усіх доступних областей України. Зміна області автоматично оновить дані у вкладках.
* **Інтервал років (Slider):** Повзунок з двома кінцями для вибору початкового та кінцевого року аналізу.
* **Інтервал тижнів (Slider):** Повзунок для вибору діапазону тижнів (від 1 до 52).
* **Сортування (Checkboxes):** Дозволяє відсортувати таблицю за обраним індексом за зростанням або за спаданням. 
  * *Увага:* Програма має вбудований захист від колізій. Якщо випадково увімкнути обидва чекбокси одночасно, з'явиться попередження, а сортування буде скасовано до виправлення вибору.
* **Кнопка "Скинути всі фільтри" (Reset):** Миттєво повертає всі повзунки, списки та чекбокси до їх початкових (дефолтних) значень.

### 2. Головне вікно 
Головне вікно розбите на три інтерактивні вкладки для зручного перегляду результатів:
* **Таблиця даних:** Відображає "сирі" дані, відфільтровані за обраними в сайдбарі роками, тижнями та областю. Таблиця реагує на чекбокси сортування.
* **Графік області:** Показує лінійний графік (часовий ряд) динаміки обраного індексу для вибраної області за вказаний проміжок часу. Дозволяє візуально оцінити зміни показника (графік інтерактивний: можна наближати окремі ділянки).
* **Порівняння областей:** Відображає стовпчикову діаграму, де порівнюється середнє значення обраного індексу між усіма областями України за вказаний період.
