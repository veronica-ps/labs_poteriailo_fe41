## Системні вимоги

* **ОС:** Будь-який дистрибутив Linux з підтримкою GUI (Ubuntu, Debian, Fedora, Arch Linux і т.п.).
* **Апаратне забезпечення:** Наявність веб-камери.
* **ПЗ:** `g++`, `cmake`, `make`, `libopencv-dev`.

## Білд та запуск

Виконайте наступні команди у терміналі по черзі для запуску проєкту:

1. **Встановлення залежностей:**
   ```bash
   ./preinstall.sh
   ```
2. **Збірка проекту:**
    ```bash
   ./build.sh
   ```
Далі перенесіть файл image6.png у створену папку build/, інакше режим PIP відобразить попередження.

3. **Запуск програми:**
    ```bash
    ./run.sh
    ```
## Керування програмою:

### Клавіатура (Фільтри) 
1 - Normal (звичайне відео без фільтрів)

2 - Invert (інверсія кольорів)

3 - Blur (розмиття, GaussianBlur)

4 - Canny (визначення контурів)

5 - Перемикач PIP (увімкнути/вимкнути накладання картинки image6.png)

ESC - Вихід з програми

### Миша (Малювання): 
ЛКМ (затиснути і тягнути): Намалювати інтерактивну стрілку.

ПКМ (клік): Стерти намальовану стрілку.

### Інтерфейс:
Слайдер Brightness: Дозволяє плавно налаштовувати яскравість відео (від -50 до +50 відсотків базового значення).
