## Архітектура проекту (Класи та модулі)
Програма побудована за модульним принципом, де кожен клас відповідає за окрему логічну частину системи:

* **`CameraProvider`**: Абстракція над `cv::VideoCapture`. Відповідає за стабільне захоплення кадрів з веб-камери.
* **`FaceDetector`**: Найважливіший модуль 7 лаб. роботи. Працює в окремому фоновому потоці (`Worker Thread`). Використовує модуль `cv::dnn` для інференсу нейромережі. Взаємодія з основним потоком синхронізована за допомогою `std::mutex` та `std::condition_variable`.
* **`FrameProcessor`**: Містить алгоритми візуальної обробки (інверсія, розмиття, Canny). Також відповідає за режим Picture-in-Picture (накладання логотипа).
* **`KeyProcessor`**: Обробляє введення з клавіатури, керує станами (`enum ProcessMode`) та перемикає активність PIP-режиму.
* **`Display`**: Відповідає за створення вікна та коректне відображення обробленого відеопотоку.

## Багатопотоковість та синхронізація
Для запобігання фризів інтерфейсу під час роботи "важкої" нейромережі реалізовано наступну схему:
1.  **Main Thread (UI)**: Читає кадри, накладає легкі фільтри та миттєво відображає відео.
2.  **Worker Thread**: Отримує копію кадру, виконує детекцію обличчя та оновлює координати.
3.  **Синхронізація**: Використовується `std::lock_guard` для безпечного доступу до спільних ресурсів та `std::atomic` для керування життєвим циклом потоку.

## Білд та запуск

Виконайте наступні команди у терміналі по черзі для запуску проєкту:

1. **Встановлення залежностей:**
   ```bash
   ./preinstall.sh
   ```
2. **Збірка проекту:**
    ```bash
   ./build.sh
   ```
Далі перенесіть файл image01.png у створену папку build/, інакше режим PIP відобразить попередження.

3. **Запуск програми:**
    ```bash
    ./run.sh
    ```
## Керування програмою:

### Клавіатура (Фільтри)
1 - Normal (звичайне відео без фільтрів)

2 - Invert (інверсія кольорів)

3 - Blur (розмиття, GaussianBlur)

4 - Canny (визначення контурів)

5 - Перемикач PIP (увімкнути/вимкнути накладання картинки image6.png)

F - Face Detection (детекція обличчя за допомогою нейромережі)

ESC - Вихід з програми

### Миша (Малювання):
ЛКМ (затиснути і тягнути): Намалювати інтерактивну стрілку.

ПКМ (клік): Стерти намальовану стрілку.

### Інтерфейс:
Слайдер Brightness: Дозволяє плавно налаштовувати яскравість відео (від -50 до +50 відсотків базового значення).
